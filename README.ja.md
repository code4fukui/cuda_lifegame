# cuda_lifegame

コンウェイのライフゲーム（Conway's Game of Life）のシンプルなCUDAベースの実装です。このプロジェクトは、NVIDIA GPU上で実行することでセル・オートマトンのシミュレーションを高速化します。

シミュレーションは事前に定義されたR-ペントミノパターンから開始し、各イテレーションごとにグリッドの状態をコンソールに出力します。

### 出力例
```
Iteration 0 (Initial R-pentomino):
00000000000000000000
00000000000000000000
00100000000000000000
00010000000000000000
01110000000000000000
00000000000000000000
...

Iteration 1:
00000000000000000000
00000000000000000000
00000000000000000000
01010000000000000000
00110000000000000000
00100000000000000000
...
```

## 機能
- **GPUアクセラレーション:** CUDAを利用してGPU上でシミュレーションを実行します。
- **固定グリッド:** トーラス型（ラップアラウンド）境界を持つ20x20のグリッドをシミュレーションします。
- **事前定義パターン:** ハードコードされたR-ペントミノパターンで初期化します。
- **固定イテレーション:** 100回のイテレーションを実行し、各ステップでグリッドの状態を出力します。

## 要件
- CUDA対応のNVIDIA GPU
- NVIDIA CUDA Toolkitがインストールされていること

## 使用方法

### 1. コンパイル
NVIDIA CUDAコンパイラ（`nvcc`）を使用してソースコードをコンパイルします。

```sh
# -Xcompiler フラグは、一部のWindowsツールチェーンで特定の警告を抑制します
nvcc -Xcompiler "/wd 4819" lifegame.cu
```
Windowsでは、付属のバッチスクリプトを使用することもできます:
```sh
c.bat
```

### 2. 実行
コンパイルされたプログラムを実行します。出力ファイル名は、Windowsでは `a.exe`、Linux/macOSでは `a.out` になります。

**Windowsの場合:**
```sh
a.exe
```

**Linux / macOSの場合:**
```sh
./a.out
```

プログラムは、100回のイテレーションにわたり20x20グリッドの状態をコンソールに出力します。
